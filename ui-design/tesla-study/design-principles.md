# 金的设计学习笔记

## 特斯拉设计核心原则

### 1. 产品即英雄（Product is the Hero）
- 每个section都是100vh全屏
- 产品图片铺满整个视口，不留灰色空白
- 图片是唯一的视觉焦点，没有多余的装饰元素
- 文字极少，只做点睛之用

### 2. 极简信息层级
- 每个section只传达一个信息
- 标题：产品名（大号，粗体）
- 副标题：一句话价值主张
- 按钮：最多2个（主CTA + 次CTA）
- 没有段落文字，没有解释，让图片说话

### 3. 按钮设计
- 主按钮：蓝色实底 #3E6AE1，白色文字
- 次按钮：白色半透明 rgba(255,255,255,0.65)，深色文字
- 暗背景上：白色轮廓按钮
- 圆角很小（4px），不是全圆角
- 间距适中，不紧不松

### 4. 导航栏
- 超轻量：高度约52-56px
- 半透明毛玻璃效果
- LOGO左侧，链接居中，工具右侧
- 字号小（14px），字重中等（500）
- 没有border-bottom阴影，干净到极致

### 5. 滚动叙事（Scroll Storytelling）
- 每滚动一屏 = 一个完整的产品故事
- 用户从上到下浏览 = 品牌故事的线性叙述
- 不用手动点击切换，滚动就是导航

### 6. 色彩克制
- 几乎不用色彩装饰
- 背景要么是产品照片，要么是纯白/浅灰
- 唯一的"颜色"是蓝色按钮
- 让产品自己的颜色和材质成为视觉焦点

## 奢侈品/高端品牌网站设计原则

### 1. 留白即奢华（White Space = Luxury）
- 越多留白越高级
- 信息密度低 = 从容感 = 高端感
- 拥挤 = 廉价感

### 2. 高质量视觉是一切
- 图片质量直接决定网站档次
- 宁可少用图，也不用差图
- 场景化展示（产品在使用环境中）比白底抠图高级100倍

### 3. 排版即品牌
- 字体选择传达品牌调性
- 字号层级要清晰（标题/副标题/正文）
- 字间距和行高也是设计的一部分

### 4. 一致性
- 每个页面遵循同样的视觉语言
- 间距、字号、颜色在全站统一
- 打破一致性 = 打破信任

### 5. 讲故事，不讲参数
- 先打动情感，再给信息
- 用户先看到的是"这个产品很美/很强大"的感觉
- 然后才是技术细节

## 我之前犯的错

1. **产品图太小** — 应该铺满整个section
2. **灰色背景块太多** — 应该用场景大图替代
3. **文字太多** — 特斯拉每个section最多20个字
4. **按钮样式不对** — 应该用蓝色实底+白色半透明
5. **信息密度太高** — 一个section塞了太多内容
6. **没有场景化** — 产品脱离了使用环境
7. **不够克制** — 加了太多装饰性元素

## 重做特斯拉版的行动计划

1. Hero：全屏产品大图，文字居顶，按钮居底
2. 每款产品一个全屏section，用AI生成高质量场景大图
3. 导航栏：52px高，毛玻璃，极简
4. 按钮：蓝色#3E6AE1 + 白色半透明
5. 2列网格：用场景大图填充，左下角白色文字
6. 轮播指示器：小圆点
7. 文字极简：每个section不超过20个英文单词
